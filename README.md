# Трекер изучения технологий

React-приложение для отслеживания прогресса изучения различных технологий с современным **GlassMorphism** дизайном в лавандово-черной цветовой палитре.

## Установка

```bash
npm install
```

Если возникают проблемы с установкой, попробуйте:

```bash
npm install --legacy-peer-deps
```

## Запуск

```bash
npm start
```

Или используйте готовый скрипт:

```bash
start.bat
```

Приложение откроется в браузере по адресу [http://localhost:3000](http://localhost:3000)
**Важно:** Убедитесь, что версия `react-scripts` в `package.json` равна `5.0.1` (не `^0.0.0`)

## Структура проекта

- Компонент `TechnologyCard` - карточка технологии с отображением статуса
- Компонент `ProgressHeader` - заголовок с прогресс-баром
- Тестовые данные в `App.js`
- Интерактивность карточек (изменение статуса по клику)
- Компонент `QuickActions` - быстрые действия
- Компонент `StatusFilter` - фильтрация по статусу
- Компонент `Statistics` - статистика
- Автосохранение в `localStorage`
- Компонент `TechnologyNotes` - заметки к технологиям
- Компонент `SearchBar` - поиск по технологиям
- Кастомный хук `useTechnologies` - логика работы с технологиями
- Переиспользуемые компоненты: `ProgressBar`, `Modal`
- Экспорт данных в консоль
- React Router для навигации
- Страницы:
  - `Home` - главная страница
  - `TechnologyList` - список всех технологий
  - `TechnologyDetail` - детальная информация о технологии
  - `AddTechnology` - добавление новой технологии
  - `StatisticsPage` - страница статистики с графиком
  - `Settings` - настройки приложения
- Кастомный хук `useTechnologiesApi` - имитация работы с API
- Компонент `RoadmapImporter` - импорт технологий из внешнего источника
- Поиск с debounce через хук `useDebounce`
- Компонент `TechnologyResources` - загрузка дополнительных ресурсов
- Компонент `Footer` - футер с навигацией и контактами
- Расширенные настройки с опциями отображения, уведомлений и статистикой

## Основные возможности

- ✅ **Современный дизайн** - вдохновленный портфолио-стилем с GlassMorphism эффектами и мягкой пастельной палитрой
- ✅ **Категории технологий** - Frontend, Backend, DevOps, Data Science, ML-dev
- ✅ **113 технологий** - тщательно отобранный список популярных и востребованных технологий
- ✅ **Фильтрация по категориям** - удобный поиск по категориям с подсчетом
- ✅ **Отслеживание статуса** - не начато, в процессе, завершено
- ✅ **Заметки** - добавление заметок к каждой технологии
- ✅ **Поиск с debounce** - быстрый поиск по названию, описанию и категории
- ✅ **Статистика** - детальная статистика по категориям и статусам
- ✅ **Визуализация прогресса** - графики и прогресс-бары
- ✅ **Автосохранение** - автоматическое сохранение в localStorage
- ✅ **Импорт дорожных карт** - импорт технологий из внешнего источника
- ✅ **Навигация** - удобная навигация между страницами
- ✅ **Экспорт/импорт данных** - резервное копирование данных (буфер обмена и файлы)
- ✅ **Расширенные настройки** - настройки отображения, уведомлений, автосохранения
- ✅ **Статистика** - детальная статистика использования в настройках
- ✅ **Доступность** - поддержка клавиатурной навигации и screen readers
- ✅ **Footer с контактами** - контактная информация разработчика
